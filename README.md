# Hotel-Booking-analysis-Dashboard
Project Title:
🏨 Hotel Booking Cancellation Analysis Dashboard

Short Description:

This project is an interactive Excel dashboard that analyzes hotel booking and cancellation patterns. It helps understand customer behavior, booking trends, and key factors affecting cancellations using visual insights.

2️⃣ 🛠️ Technologies Used

Microsoft Excel
Pivot Tables (data summarization ke liye)
Pivot Charts (visualization ke liye)
Slicers (interactive filtering – year wise)
Dashboard Design (layout & UI)
Data Cleaning (basic preprocessing)
3️⃣ 📁 Data Source
Hotel Booking Dataset (2015–2017)
Public dataset (Kaggle)
Data includes:
Booking details
Hotel type (City / Resort)
Guest type (Couple, Family, Single)
Room status
Cancellation status
4️⃣ ⭐ Features / Highlights
📅 Year-wise filter (2015, 2016, 2017)
🏨 Total bookings by hotel type
❌ Cancellation analysis (City vs Resort)
👨‍👩‍👧 Guest type vs cancellation comparison
🛏 Room status vs cancellation
📈 Monthly booking vs cancellation trend
📊 KPI cards:
Total Bookings
Total Cancellations
🎯 Clean & interactive dashboard design
5️⃣ 📊 Business Impact & Insights
City Hotels show higher cancellations → need better booking policies
Couples are the major contributors to bookings & cancellations
“Desired Room” bookings also have high cancellations → expectation mismatch
Peak bookings occur in mid-year (seasonal demand)
Helps hotels:
Improve customer retention
Optimize pricing strategy
Reduce cancellation rate
Better demand forecasting
